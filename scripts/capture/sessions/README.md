# sessions/

Canonical ANSI-colored fixture files for `capture-cli.sh`. Each `.ansi` file
is a representative snapshot of a single Claude Code moment — not a live
recording. Live recordings drift; fixtures are deterministic.

Generated by `../generate-fixtures.sh`. Edit the generator, not the output.

## Naming

`chN-topic.ansi` — one shot per MDX callsite. Matches output PNG name under
`public/screenshots/`.

## Why fixtures instead of live capture

Claude Code's TUI is interactive and animated. Capturing `claude` live via
`tmux capture-pane` is possible but fragile: timing-sensitive, machine-state-
dependent, non-reproducible across contributors. Hand-crafted ANSI fixtures
give us pixel-stable PNGs that survive Claude Code version bumps and
contributor machines.
